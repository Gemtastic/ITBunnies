# ITBunnies
This is my first attempt at implementing a RESTful API a la HATEOAS and a SOAP API.

This application lets you handle quotes, product types and comments on quotes made to the IT bunnies.

###Tech Spec:
Database:   PostgreSQL
ORM:        JPA
Java:       JavaEE 7
WebService: JAX-RS
            JAX-WS
            JAX-B
